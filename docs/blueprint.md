# **App Name**: Rajasthan Roots

## Core Features:

- Interactive Village Map: Display a clickable village map with hotspots for each level, providing a visual and interactive learning experience.
- Level Content Display: Load and display engaging, structured educational content about the Cobra Gypsy Tribe for each level, including text, images, and videos.
- Dynamic Quiz Generation: Generate quizzes by randomly selecting questions from a Firestore question bank to test user knowledge at each level.
- Progress Tracking: Store and display user progress, including completed levels, scores, and overall progress percentage, to encourage continued learning.
- Content Summarization Tool: Generate summaries of the content using a Large Language Model tool. Summarization will only be triggered for lessons which the LLM reasons the tool can successfully produce an effective, educational summarization. These LLM-generated summaries may be available to the user before, during or after completing the standard reading activities for the lessons, based on whatever the LLM determines to be most beneficial pedagogically.
- User Authentication: Allow users to register and log in using email/password or Google Sign-in, securely managing user data and progress.
- Content Generation Tool: Leverage an LLM as a tool to automatically generate additional questions for quizzes based on current level content.

## Style Guidelines:

- Primary color: Vibrant orange (#FF8C00) to represent the energy and warmth of Rajasthan.
- Background color: Desaturated orange (#F2E0D1) for a soft, earthy tone.
- Accent color: Analogous yellow-orange (#FFA500) for highlights and call-to-action buttons.
- Headline font: 'Playfair' (serif) for an elegant, high-end feel.
- Body font: 'PT Sans' (sans-serif) paired with Playfair, for clear and readable content.
- Use culturally relevant icons and symbols, reflecting Rajasthani art and culture.
- One-page layout with hotspots on a village image; quiz pop-up modal.
